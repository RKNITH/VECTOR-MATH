Vector Applications in Mathematics

History of Vectors

Vectors have a rich history in mathematics and physics, evolving over centuries. Their development has been pivotal in various scientific fields, shaping our understanding of the physical world and advancing technology.

  • Ancient Greece:
    • Concepts of magnitude and direction were used in geometry, particularly in the works of Euclid and Archimedes.
    • Early ideas of vectors can be seen in the use of geometric representations of physical quantities.
  • 17th Century:
    • Isaac Newton's work in mechanics introduced the concept of force as a vector quantity.
    • Vector ideas were implicit in Newton's laws of motion and gravitation.
  • 19th Century:
    • William Rowan Hamilton developed the theory of quaternions, a precursor to vector algebra.
    • Hermann Grassmann's work on linear algebra laid the foundation for modern vector spaces.
    • Josiah Willard Gibbs and Oliver Heaviside formulated vector calculus, which became essential for electromagnetic theory.
  • 20th Century:
    • Vectors became extensively used in physics, particularly in quantum mechanics, relativity, and electromagnetism.
    • The introduction of tensors, which generalize vectors, had a profound impact on the theory of relativity.
    • Vectors and vector spaces became fundamental concepts in computer graphics, engineering, and computer science.
  • 21st Century:
    • Advancements in technology and computational power have enabled complex vector calculations and simulations in various fields.
    • Vectors play a crucial role in machine learning, data science, and artificial intelligence, particularly in the representation of high-dimensional data.
    • Ongoing research continues to expand the applications of vectors in new and emerging scientific and engineering disciplines.

Definition of Vectors

A vector is a quantity having both magnitude and direction. It is usually represented by an arrow where the length represents the magnitude and the arrowhead indicates the direction. Vectors are fundamental in physics and engineering as they describe quantities that have both size and direction.

Examples: Displacement, velocity, force.

Vector notation: v = (v1, v2, v3)

In a three-dimensional space, a vector can also be represented using the unit vectors i, j, k, which point in the direction of the x, y, and z axes respectively:

v = v1i + v2j + v3k

Here, i, j, k are unit vectors along the x, y, and z axes:

  • i: Unit vector along the x-axis
  • j: Unit vector along the y-axis
  • k: Unit vector along the z-axis

These unit vectors have a magnitude of 1 and are orthogonal to each other, providing a basis for representing any vector in three-dimensional space.

For example, the vector v = 3i + 2j + 1k has components 3, 2, and 1 along the x, y, and z axes respectively. This means the vector points 3 units in the x direction, 2 units in the y direction, and 1 unit in the z direction.

Types of Vectors

Vectors can be classified into different types based on their properties and applications:

  • Zero Vector: A vector with zero magnitude and no specific direction. Represented as 0 or (0, 0, 0) in three-dimensional space.
  • Unit Vector: A vector with a magnitude of one, used to indicate direction. Represented as i, j, k in Cartesian coordinates.
  • Position Vector: A vector that specifies the position of a point relative to an origin. Represented as r = (x, y, z).
  • Equal Vectors: Vectors that have the same magnitude and direction, regardless of their initial points.
  • Negative Vectors: Vectors that have the same magnitude but opposite direction to a given vector. If a is a vector, -a is its negative vector.
  • Parallel Vectors: Vectors that have the same or opposite direction. They are either in the same line or in parallel lines.
  • Orthogonal Vectors: Vectors that are perpendicular to each other. Their dot product is zero.
  • Coplanar Vectors: Vectors that lie in the same plane.
  • Collinear Vectors: Vectors that lie along the same line, either in the same or opposite direction.

Linearly Dependent and Independent Vectors

Linearly Dependent Vectors: Vectors that can be expressed as a scalar multiple of each other.

Linearly Independent Vectors: Vectors that cannot be expressed as a scalar multiple of each other.

Example: Checking linearity of vectors v1 = (1, 2, 3) and v2 = (2, 4, 6).

If v2 = kv1, then they are linearly dependent. Here, k=2.

Collinearity of Vectors

Two or three vectors are collinear if they lie along the same line.

Example: Vectors a = (1, 2) and b = (2, 4) are collinear.

Condition for collinearity: b = ka for some scalar k.

Coplanarity of Vectors

Three or four vectors are coplanar if they lie in the same plane.

Example: Vectors a = (1, 2, 3), b = (4, 5, 6), and c = (7, 8, 9) are coplanar if their scalar triple product is zero.

Condition for coplanarity: a · (b × c) = 0

Dot & Cross Products

Dot Product: The scalar product of two vectors.

Formula: a · b = |a| |b| cosθ

Example: a = (1, 2, 3) and b = (4, 5, 6).

a · b = 1×4 + 2×5 + 3×6 = 32

Cross Product: The vector product of two vectors.

Formula: a × b = |a| |b| sinθ n

Example: a = (1, 2, 3) and b = (4, 5, 6).

a × b = (2×6 - 3×5, 3×4 - 1×6, 1×5 - 2×4) = (-3, 6, -3)

Reciprocal System

Reciprocal vectors are vectors that satisfy certain orthogonality conditions with the original set of vectors. They are particularly useful in crystallography, physics, and engineering for analyzing wave vectors, lattice structures, and more.

Formula: If {a, b, c} is a set of vectors, the reciprocal set {α, β, γ} is defined such that:

  • α · a = 1, α · b = 0, α · c = 0
  • β · a = 0, β · b = 1, β · c = 0
  • γ · a = 0, γ · b = 0, γ · c = 1

These conditions ensure that each reciprocal vector is orthogonal to the other two original vectors and has a dot product of 1 with its corresponding original vector.

To calculate the reciprocal vectors, we use the following formulas in i, j, k notation:

If a = a1i + a2j + a3k, b = b1i + b2j + b3k, c = c1i + c2j + c3k, then:

  • α = (b2c3 - b3c2)i + (b3c1 - b1c3)j + (b1c2 - b2c1)k / V
  • β = (c2a3 - c3a2)i + (c3a1 - c1a3)j + (c1a2 - c2a1)k / V
  • γ = (a2b3 - a3b2)i + (a3b1 - a1b3)j + (a1b2 - a2b1)k / V

Where V is the volume of the parallelepiped formed by vectors a, b, and c and is given by:

V = a · (b × c)

Example:

Consider the vectors a = 2i + 3j + k, b = i + 2j + 3k, c = 3i + j + 2k. The reciprocal vectors can be calculated as:

  • α = ((2*2 - 3*1)i + (3*3 - 1*3)j + (1*1 - 2*2)k) / V
  • β = ((1*2 - 2*3)i + (3*3 - 2*3)j + (2*1 - 1*2)k) / V
  • γ = ((3*2 - 1*3)i + (1*1 - 2*3)j + (2*2 - 1*3)k) / V

Where V = 2(2*2 - 3*1) - 3(1*2 - 3*3) + 1(1*1 - 3*2).

Vectors in Straight Line

Vectors are fundamental in understanding and calculating various aspects of straight lines in mathematics.

Formulas and Examples:

Equation of a Straight Line:

Vector form: r = a + λb

Example: If a = (1, 2, 3) and b = (4, 5, 6), the equation is r = (1, 2, 3) + λ(4, 5, 6).

Bisector of Angle:

Formula: If a and b are two vectors, the bisector is given by (|a|b + |b|a)/( |a| + |b|).

Example: For a = (1, 2, 2) and b = (2, 2, 1), bisector = (3(2,2,1) + 3(1,2,2))/6 = (7/6, 8/6, 7/6).

Sectional Formula:

Formula: If A and B are two points and P divides AB in the ratio m:n, then P = (mB + nA)/(m+n).

Example: For A = (1, 2) and B = (3, 4), if P divides AB in the ratio 1:1, then P = (1(3,4) + 1(1,2))/2 = (2, 3).

Perpendicular Distance Formula:

Formula: Distance from point P(x1, y1) to the line Ax + By + C = 0 is given by |Ax1 + By1 + C| / √(A² + B²).

Example: For point (1, 1) and line 3x + 4y - 5 = 0, distance = |3(1) + 4(1) - 5| / √(3² + 4²) = 2 / 5 = 0.4.

Skew Lines

Skew lines are lines that do not intersect and are not parallel. They do not lie in the same plane.

Example: Lines l1: r = a + λb and l2: r = c + μd are skew if there is no common solution for λ and μ.

Shortest Distance Between Skew Lines:

Formula: d = |(a - c) · (b × d)| / |b × d|

Example: For lines l1: r = (1, 2, 3) + λ(4, 5, 6) and l2: r = (7, 8, 9) + μ(10, 11, 12), the shortest distance is calculated as follows:

  • a - c = (1 - 7, 2 - 8, 3 - 9) = (-6, -6, -6)
  • b × d = (4, 5, 6) × (10, 11, 12) = (30 - 36, 24 - 60, 44 - 50) = (-6, -36, -6)
  • |b × d| = √((-6)² + (-36)² + (-6)²) = √(6×6 + 36×36 + 6×6) = √(6×6(1 + 6×6)) = 6√(1 + 36) = 6√37 = 36√37 = 12
  • d = |(a - c) · (b × d)| / |b × d| = |-6(1) + -6(6) + -6(0)| / |6√37| = 0.6

Therefore, the shortest distance is 6√37.
